# 🍔 Hungry Hare Food Ordering App

A beginner full-stack project built using Python, JavaScript, and SQLite.

## Features
- CLI-based food ordering system (Python)
- Web-based UI (HTML, CSS, JS)
- Cart management with quantity
- Login system (frontend)
- Order history (localStorage)
- SQLite database integration

## Tech Stack
- Python
- JavaScript
- SQLite
- HTML/CSS

## Future Improvements
- Flask backend API
- Real authentication system
